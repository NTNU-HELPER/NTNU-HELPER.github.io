# 🎯 Guess the Number

一款簡單又上癮的數字猜謎遊戲，挑戰你的直覺與邏輯思維！  
適合所有年齡層，無需網路連線，隨時隨地開玩！

## 🧠 遊戲玩法

- 系統會隨機產生一個數字（預設範圍為 1 到 100）。
- 玩家每次輸入一個數字作為猜測。
- 系統會回應：
  - 數字太大
  - 數字太小
  - 恭喜猜中！
- 玩家可以不斷嘗試直到猜中正確答案。
- 支援重置與重新開始遊戲。

## 🛠️ 技術說明

- 開發工具：HTML / CSS / JavaScript（或使用原生 App 技術，如 Flutter、React Native 等）
- 平台支援：Web / Android / iOS
